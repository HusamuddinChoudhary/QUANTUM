# QUANTUM-ARES

QUANTUM-ARES is a cybersecurity infrastructure validation SaaS specializing in Zero-Trust, Quantum Risk, and Attack Path analysis.

## 🏗️ Architecture

- **Frontend**: React 18, Vite, Three.js, Cytoscape
- **Backend**: FastAPI 0.115+, SQLAlchemy 2.0 (Async)
- **Database**: PostgreSQL 16 (or local SQLite/Mock)
- **Task Queue**: Celery 5 + Redis 7
- **Security Engines**: NetworkX Graph Engine, QVI Analysis
